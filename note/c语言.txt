
	1.C语言入门	
		-ANSI C一共只有32关键字
			auto break...
		-运算符 34种
		-数据结构类型丰富
		-具有结构化的控制语句
		-语法限制不严格,设计自由度大
		-C语言允许直接访问物理地址,能进行为操作
		-生成目标代码质量高,程序执行效率高
		-相比汇编,可移植性好

	2.面向过程,面向对象设计思想
		-面向过程：以事件为中心,先分析解决问题的需要的步骤,用函数一步步实现,依次调用
		-面向对象：类似于已经将函数封装好了,只需要致力于功能的实现即可。

	3.C主函数main
		例：
			#include<studio.h>
			void main()
			{
				printf("世界,你好！\n")
			}
		-函数都只有一个main函数，为主函数，为程序的入口。
		
	4.数据类型：
		-基本类型：
			-数据类型分为：整型,浮点型,字符型,枚举型
			-数据是否改变分为：常量,变量。
			-常量：
				#define 标识符 常量
				-例：	
					#define CAT 1
			-变量：
				数据类型 变量名 = 变量值
				-例：
					int k = 3
			*常量都是大写,变量都是小写
			-整型：
				-除10进制外其余进制均已前缀代表其进制
				-八进制：以0开头,通常为无符号数
				-十六进制：前缀为0x
		
		
		-构造类型：
			-分为：数组类型,结构体类型,共用体类型
		
		
		
		
		-指针类型：
		
		-空类型：
			void,函数没有返回值,定义为空类型。
