 
 1.linux常用终端命令
	1. 放大窗口 ctrl + ; 
	2. 缩小窗口 ctrl - ;
	3. 清屏  clear;
	4. 查看当前文件夹的内容 ls;
		- -a 显示所有文件

	8. 创建目录 mkdir[目录名];

	9. 删除指定文件 rm[文件名];
		- -r删除文件夹包括其所有内容

    22.&&, &, |, ||：
        &   表示任务在后台执行,如要在后台运行redis-server,
            eg：  redis-server &
        &&  表示前一条命令执行成功时,才执行后一条命令 ,
            eg： echo '1‘ && echo '2'    
        |   表示管道,上一条命令的输出,作为下一条命令参数,
            eg： echo 'yes' | wc -l
        ||  表示上一条命令执行失败后,才执行下一条命令
            eg：cat nofile || echo "fail"

  4.远程管理常用命令
	-关机\重启  shutdown 选项 时间
		// shutdown -r now 现在重新启动
		-r 重新启动
		-c 取消
	-查看或配置网卡信息
		-ifconfig 查看/配置计算机当前的网卡配置信息
		-ping ip地址  检查ip地址是否连接正常
			-ctrl + c 结束检查
	-远程登录和复制文件
		-服务器端口 SSH服务器 22
					web服务器 80
					HTTPS    443
					FTP服务器 21
		-ssh [-p 端口号] user@remote
			-p 默认端口22可以不输入 ,
			-user 远程机器上的用户名,如果不指定就默认当前用户名
			-remote 是远程机器的地址,有IP/域名/别名
		-scp 远程拷贝文件
			-scp -P port 文件名 user@remote:Desktop/文件名
			 将本地文件拷贝到远程家目录的Desktop文件下
			*如果后面跟的不是绝对路径,则以用户的家目录作为参照目录
			
			-scp -P port user@remote:Desktop/文件名 文件名
			 将远程文件拷贝到本地当前目录上
			-r  若原文件是目录  则复制整个目录
			*Windows 用ssh文件要:putty、pscp；或者Filezilla进行FTP传输端口是21
		-ssh高级
			-~/.ssh  保存ssh配置信息 
			-免密登录
				-ssh-keygen 生成SSH钥匙
				-ssh-copy-id user@remote
				-其生成的配置文件名有两种：
						-id_rsa_pub  公钥
						-id_rsa      私钥
						-本地使用私钥加密/解密
						-服务器使用公用加密/解密
						-非对称加密算法
			-配置别名
				-在ssh创建config文件
				-在config文件中输入：
					Host 别名
						HostName ip地址
						User     用户名
						port     端口
		-exit 退出

  5.权限,查询命令	
	-用户权限命令		
		-权限有：读    r  red	  4
				 写    w  write   2
				 执行  x  excute  1
		 例：drwxr-xr-x 2 ltw ltw 4096 7月 23 09:42 下载
		     其中 rwx|r-x|r-x  代表 用有者|组|其他用户 的权限	
		     后面跟的数字 2 代表其硬链接数	
		-硬连接数：有多少种方式可以访问到当前文件/目录	
		           文件基本为1,目录取决与其子目录数量
		-chmod 修改用户/组 对文件/目录的权限
		   格式：chmod +/-rwx 文件名|目录名  
				//chmod +rwx  123.txt  让123.txt拥有rwx权限
		-执行文件： ./文件名	
		-超级用户 sudo命令进入超级用户
	-组的管理
		-添加/删除/确认组的信息
			-groupadd 组名     添加组
			-groupdel 组名     删除组
			-cat /etc/group    确认组的信息
			-chgrp 组名 文件/目录名   修改文件/目录的所属组
				-R 递归修改(目录下面有其他文件时,)
		-与群组相关文件保存地址：/etc/group
	-用户管理
		-userad 新建用户名                添加新用户
			-m 自动建立用户家目录
			-g 组 指定用户所在的组(默认同名组) 
		-password 用户名                  设置用户密码
		-userdel -r 用户名                删除用户
			-r 会自动删除用户家目录
		-cat /etc/passwd | grep 用户名    确定用户信息
		-与用户相关的文件保存地址：/etc/passwd
		-修改用户密码的程序： /user/bin/passwd
			-用户名:密码:uid:gid:全名:家目录
		-查看用户信息
			-id [用户名]    查看用户uid和gid信息
			-who		    查看当前所有登录的用户列表
			-whoami         查看当前登录的账号名
		-usermod 组 用户名      设置用户的组
			-g  修改主组
			-G  修改附加组
		-usermod -s /bin/bash 用户名  修改用户登录Shell
			-shell 就是终端命令 ubuntu指定的是dash(简洁高速)
		*Which 查看执行命令所在位置
			-/bin  二进制执行文件目录,主要具体应用
			-/sbin 系统管理员二进制代码目录,只要用于系统管理
			-/usr/bin    后期安装的软件
			-/usr/sbin	  超级用户的一些管理程序
		-切换用户su
			-su - 用户名  切换用户,(因为有-号)同时切换到~/
			-exit        退出当前登录账号
		-修改文件权限
			-chown   修改拥有者
			格式: chown 用户名 文件名|目录名
			-chgrp   递归修改组
			格式：chgrp -R 组名 文件名|目录名

  10. linux网络相关
    -Ubuntu18.0.4以上版本设置静态ip:
        在 /etc/netplan/xxx.yml 文件下修改:
            network:
              version: 2
              renderer: networkd
              ethernets:
                ens33:
                  addresses: [192.168.56.11/24]
                  gateway4: 192.168.56.2
                  nameservers:
                    addresses: [192.168.56.2, 8.8.8.8]
                  dhcp4: no
                  optional: no
        应用至ip地址:
            sudo netplan apply
    -查看ip、网关、子网掩码等情况
        route -n
    -建立bind9 DNS服务器
        -下载:
            sudo apt install bind9
        -启动：
            sudo systemctl start bind9
        -停止:
            sudo systemctl stop bind9
        -重启：
            sudo systemctl restart bind9
        -状态：
            sudo systemctl status bind9
